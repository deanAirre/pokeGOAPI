# PokemonGO API

A Go REST API to make a working API that fetch public API https://pokeapi.co/ to give RESTFUL response to React Vite frontend

This API uses Go and PostgreSQL stack, to use this API, readers must at least preinstall things listed below:

1. `postgresql-server` and `postgresql-contrib`
2. Golang version go1.25.5, tested using `go1.25.5 X:nodwarf5 linux/amd64`

through means suitable with user's OS is it Windows or Linux
